## Запуск проекта

### 1. Клонировать/обновить репозиторий

Клонировать
```bash
git clone <ссылка_на_репозиторий>
cd frontend
```
Обновить
```bash
git pull
cd frontend
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Запустить проект

```bash
npm run dev
```

После запуска открой в браузере:

```
http://localhost:5173
```
