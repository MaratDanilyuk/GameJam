🧩 Лоскутная реальность
Прототип веб-игры с фиджитал-механикой для детей 6–11 лет
________________________________________
🚀 Идея проекта
«Лоскутная реальность» — это веб-игра, в которой действия ребёнка в реальном мире напрямую влияют на игровой мир.
Игрок взаимодействует с персонажем — Хранителем Потерянных Историй, существом, чьё состояние отражает разрушенный мир. Чтобы восстановить его и окружающую реальность, ребёнку нужно выполнять простые, но полезные офлайн-действия: читать, двигаться, рисовать, исследовать окружающий мир.
Каждое действие вне экрана превращается в изменение внутри игры.
________________________________________
❗ Проблема
Современные дети проводят большое количество времени перед экраном, при этом:
•	снижается уровень физической активности
•	уменьшается вовлечённость в творческие и развивающие занятия
•	цифровые продукты редко мотивируют ребёнка выйти в реальный мир
•	родителям сложно оценить реальную пользу от времени, проведённого в приложении
Большинство игр удерживают внимание внутри экрана, но не создают связи с реальной жизнью.
________________________________________
💡 Решение
Мы предлагаем фиджитал-подход, при котором:
👉 игра не заменяет реальность, а стимулирует активность вне экрана
Игровой процесс строится по циклу:
1.	Игрок получает задание в игровом мире
2.	Выполняет действие в реальности (чтение, движение, творчество и т.д.)
3.	Система фиксирует выполнение (таймер, звук, фото, действия)
4.	Игровой мир реагирует и восстанавливается
5.	Игрок получает награду и влияет на развитие персонажа
________________________________________
🌍 Почему это круто
1. 🔄 Связь “экран → реальность”
Игрок не просто играет — он действует в реальном мире, а игра становится мотивацией.
________________________________________
2. ❤️ Эмоциональная вовлечённость через персонажа
Хранитель Потерянных Историй — это не NPC, а отражение прогресса ребёнка:
•	был разрушен → становится целостным
•	слабый → оживает
•	меняется в зависимости от действий игрока
👉 ребёнок помогает не системе, а «живому существу»
________________________________________
3. 🎨 Реальные действия = реальные изменения мира
Каждый тип активности влияет на игру:
•	чтение → восстанавливает тексты и знания
•	рисование → возвращает цвет
•	движение → оживляет мир
•	звук → наполняет пространство жизнью
•	поиск объектов → добавляет элементы в мир
👉 это создаёт прямую причинно-следственную связь
________________________________________
4. 🧠 Развитие через игру
Игра покрывает ключевые навыки:
•	когнитивные (чтение, наблюдение)
•	творческие (рисование, креативность)
•	моторные (движение, активность)
________________________________________
5. 👨‍👩‍👧 Прозрачность для родителей
Родитель видит:
•	какие активности выполнял ребёнок
•	сколько времени он потратил на полезные действия
•	как это повлияло на развитие и игровой мир
👉 не “время в телефоне”, а результат развития
________________________________________
🧩 Ключевая механика
Игровой мир разделён на локации, каждая из которых связана с типом активности:
•	📚 Разорванная Библиотека — чтение
•	🎨 Стертая Мастерская — рисование
•	🏃 Замерший Город — движение
•	🔊 Тихая Долина — звук
•	🔍 Мир Пропавших Объектов — исследование
Игрок восстанавливает эти локации через реальные действия.
________________________________________
🎯 Ценность проекта
Проект решает сразу несколько задач:
•	снижает пассивное потребление контента
•	превращает игру в инструмент развития
•	формирует полезные привычки
•	усиливает связь ребёнка с реальным миром
•	даёт родителям понятный инструмент контроля и поддержки
________________________________________
🏁 Итог
«Лоскутная реальность» — это не просто игра.
Это система, в которой
👉 реальные действия ребёнка становятся частью игрового мира
👉 прогресс ощущается визуально и эмоционально
👉 развитие превращается в увлекательный процесс
